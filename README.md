**Global Superstore 2016 Excel Dashboard**

**Overview-**
An end-to-end Excel data analytics project built using the Global Superstore 2016 dataset. The project covers data cleaning, KPI calculations, Pivot Table analysis, interactive dashboards, and data validation.

**Features-**
Extracted Product Sub-Categories using Flash Fill
Filled missing Postal Code and Region values using Go To Special
Calculated Order Aging using Date Functions
Built Customer Search Tool using XLOOKUP
Created Pivot Tables for Category, Market, and Monthly Sales Analysis
Added Conditional Formatting for performance highlighting
Developed an interactive Dashboard with:
Slicers (Year, Region, Segment)
Timeline Filter
Pivot Charts
Treemap/Sunburst Chart
Sales vs Profit Combo Chart
Implemented Data Validation for controlled user input
Applied worksheet protection and dashboard theming
